# Next Word Prediction using LSTM

This project implements a Next Word Prediction model using LSTM trained on a custom FAQ dataset.

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NLP (Tokenizer, Padding)

## Features
- Predicts next word based on user input
- LSTM-based sequence modeling
- Web UI using Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
